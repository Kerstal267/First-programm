# 🖥️ Tkinter GUI App


Простой графический интерфейс (GUI) на Python с использованием библиотеки Tkinter.  
Приложение позволяет вводить текст и динамически обновлять содержимое label.

---

## 📌 Описание проекта

Этот проект демонстрирует базовые возможности создания GUI-приложений на Python:

- Работа с окнами (Tk)
- Использование Frame для структуры интерфейса
- Ввод текста через Entry
- Обработка событий (нажатие кнопки)
- Динамическое обновление интерфейса

---

## ⚙️ Функциональность

✅ Ввод текста пользователем  
✅ Кнопка для обработки ввода  
✅ Динамическое изменение текста label  
✅ Изменение цвета текста после нажатия  
✅ Простая и понятная структура UI

### 1. Установи Python (если ещё нет)
Проверь:

```bash

````

### 2. Склонируй репозиторий

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### 3. Запусти приложение

```bash

```

---

## 🧠 Как это работает

1. Пользователь вводит текст в поле ввода (Entry)
2. Нажимает кнопку "Оновити лейбл"
3. Программа:

   * считывает текст
   * проверяет, пустой ли он
   * обновляет label
   * меняет цвет текста на жёлтый
